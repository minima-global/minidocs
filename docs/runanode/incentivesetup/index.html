Incentive Program Setup

After installing Minima, if you have registered for the Incentive Program you must setup your Incentive Program account to receive daily Rewards. If you have not registered, click here to sign up.

The following guidance is for Windows/Mac/Linux Desktop and Server users. Android users please see here.

There are two methods to setup your Incentive Program account:

A. Use the Incentive Program MiniDapp (recommended)
B. Use the Minima Terminal (advanced users)

A. Using the Incentive Program MiniDapp (recommended)‚Äã

  1. Download and save the Incentive Program MiniDapp from our MiniDapp site.

  2. Navigate to your MiniDapp hub:

  • Windows/Mac/Linux Desktop users, go to https://127.0.0.1:9003/ in your web browser (Firefox is not supported)
  • Server users, go to https://YourServerIP:9003/
Security warning

You may be shown a security certificate warning, to which you can click on Advanced, then Proceed.
This may be different depending on the browser and OS you are using; details on how to do this can be found here.

You will see your MiniDapp System (MDS) login page.

  1. Get your password
  • Windows/Mac/Linux Desktop users, return to the Windows/Linux Command Line or Mac Terminal where the Minima logs are running, type mds, press Enter
  • Server users, login to your server and type curl 127.0.0.1:9005/mds, press Enter

You should see an output similar to below:

mds
{
  "command":"mds",
  "status":true,
  "response":{
    "password":"1SV0-F0WN-K367",
    "minidapps":[]
  }
}
  1. Copy and paste your password into the field and click login - you will see your MiniDapp hub!
  2. To install the Incentive Program MiniDapp, click on Choose File and select the downloaded MiniDapp from your computer's downloads folder

  1. Click Install and return to the hub and open the app
  2. Follow the instructions to copy your Incentive ID from the Incentive Program website and paste into the field provided
  3. Click Update

Your Incentive account is now connected to your now and you will receive 1 Reward each day your node is running.

Using the Incentive Program app, you can:

  • See your Rewards balance by going to the Rewards tab
  • See your unique Invite link by going to the Invite Link tab. Learn more about the invite program here.

B. Using the Minima Terminal (advanced method)‚Äã

  1. Login to https://incentive.minima.global/

  2. Go to the Incentive ID page and copy your Incentive ID

  3. Enter the following command:

Windows/Mac/Linux Desktop users

  • Return to the Windows/Linux Command Line or Mac Terminal where the Minima logs are running
  • Type incentivecash uid:xxx-xxx-xxx-xxx-xxx (replacing xxx-xxx-xxx-xxx-xxx with your OWN incentive ID)
  • Press Enter

Example:

incentivecash uid:00F3E50D-5A52-444B-8F1A-0DA72D6CAA84

Server users

  • Login to your server
  • Type curl 127.0.0.1:9005/incentivecash%20uid:xxx-xxx-xxx-xxx-xxx (replacing xxx-xxx-xxx-xxx-xxx with your OWN incentive ID)
  • Press Enter

Example:

curl 127.0.0.1:9005/incentivecash%20uid:00F3E50D-5A52-444B-8F1A-0DA72D6CAA84

You should receive a return status of true and see your Reward balances.

Example:

incentivecash
{
  "command":"incentivecash",
  "status":true,
  "response":{
    "uid":"731ae11b-b602-4dea-8564-7cb4edbe07b7",
    "details":{
      "lastPing":"2022-03-14T17:28:38.709Z",
      "inviteCode":"BAEPAAD0",
      "rewards":{
        "dailyRewards":60,
        "previousRewards":100.0,
        "communityRewards":50.0,
        "inviterRewards":10.0
      }
    }
  }
}
Multiple Nodes?

If you have multiple nodes on one server, you will need to change the port number in the curl command to the next number from the custom port you chose
E.g. if you used -9121 to start your node, the RPC port will be 9125

Example:

curl 127.0.0.1:9125/incentivecash%20uid:xxx-xxx-xxx-xxx-xxx

Your Incentive account is now connected to your now and you will receive 1 Reward each day your node is running.

  1. To check your Rewards:
  • Windows/Mac/Linux Desktop users, type incentivecash
  • Server users, type curl 127.0.0.1:9005/incentivecash | jq into the Command Line on your server.
Rewards

For every day your node is connected to the network, you will receive 1 Reward within the next 24 hours.
